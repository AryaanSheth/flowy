capabilities

What it does

  • → Push-to-talk Any global hotkey. Hold to record from any app — email, code editor, browser, terminal. Release to transcribe. Text injects directly into the focused window.
  • → Native recognition Powered by Apple's SFSpeechRecognizer — built into macOS. No model files to download, no warm-up time, optimised for Apple Silicon from day one.
  • → Custom dictionary Define word substitutions that apply after every transcription. Teach Flowey your names, abbreviations, and domain jargon. Test them live in the settings panel.
  • → Ollama polish Optionally route transcriptions through a local LLM to fix punctuation, grammar, and capitalisation. Still entirely offline — no cloud round-trips.
  • → Rust + Tauri Not Electron. The binary is under 15 MB. Zero background CPU when idle. Sits silently in your menu bar until you need it.
